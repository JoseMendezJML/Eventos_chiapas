# Eventos Chiapas

## Descripción del proyecto

Eventos Chiapas es un sitio web estático desarrollado para una agencia local dedicada a la organización de eventos sociales, culturales y empresariales en Chiapas.

Este proyecto fue realizado como una práctica colaborativa utilizando Git y GitHub, aplicando ramas, commits, Pull Requests, revisión de código, resolución de conflictos, documentación y publicación mediante GitHub Pages.

## Objetivo general

Aplicar el uso de Git y GitHub en un entorno colaborativo realista, desarrollando un sitio web básico y usando buenas prácticas de control de versiones.

## Objetivos específicos

- Crear un repositorio en GitHub.
- Clonar el repositorio usando GitHub Desktop.
- Trabajar el proyecto en Visual Studio Code.
- Crear ramas para dividir el trabajo.
- Realizar commits claros y significativos.
- Crear Pull Requests hacia la rama main.
- Revisar y aprobar cambios de otros compañeros.
- Simular y resolver conflictos.
- Documentar el proceso del proyecto.
- Publicar el sitio web con GitHub Pages.

## Tecnologías utilizadas
